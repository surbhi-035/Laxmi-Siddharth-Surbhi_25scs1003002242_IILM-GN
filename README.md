Description

This project is an AI-based Text Summarizer that converts long text into a short and meaningful summary using Natural Language Processing (NLP). Technologies Used

* Python
* Streamlit
* Transformers (Hugging Face)

 How to Run the Project

1. Install required libraries:
   pip install streamlit transformers

2. Run the application:
   streamlit run summarizer_app.py

##  Project Files

* summarizer_app.py → Main code
* AI_Text_Summarizer_Report.pdf → Project report

##  Output

The application takes input text and generates a summarized version instantly.
